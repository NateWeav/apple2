Acheter OS X Server

En un clin d’œil, votre Mac devient un puissant serveur.

macOS Server démultiplie la puissance de votre entreprise, de votre bureau à domicile ou de votre établissement scolaire. Conçu spécialement pour macOS et iOS, macOS Server simplifie la configuration de Mac et des appareils iOS. Il est aussi remarquablement facile à installer, à régler et à gérer. Ajoutez-le à votre Mac depuis le Mac App Store pour seulement 27,99 $.

Votre poste de commande.

macOS Server permet de configurer et de gérer de multiples ordinateurs Mac et appareils iOS directement depuis votre Mac. Il est si convivial que vous n’aurez pas besoin d’un service des TI.
