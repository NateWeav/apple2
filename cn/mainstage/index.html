MainStage 3

舞台音效，媲美录音室效果。

现在，将你的 Mac 变身现场演出装备。凭借不计其数的各种插件和音效选择，你可以让键盘、吉他或演唱表现令观众耳目一新，难以忘怀。
从 Mac App Store 下载 MainStage 3，仅需 RMB 198。

各色表演者，都有得心应手的控制台。

MainStage 不会让任何类型的表演者失望。你可以使用来自 Logic Pro 的各种心仪乐器和特效插件，安装和拆卸都十分简单。背景伴奏让你能随心驾驭震撼的音效。你还能尽情使用心仪的硬件控制器。

键盘手，
梦幻演出蓄势待发。

现在，你可让表演效果超越实际的表演水平。Arpeggiator 拥有基于音符的远程控制和灵活的 latch 模式。而 Chord Trigger 可让你通过点按一个音符来触发一整段复杂的和弦。通过一组直观的控制，使用 Retro Synth 来重现你钟爱的 70 和 80 年代电子音效。也可将 Hammond B3 风琴、Hohner Clavinet D6 或 Fender Rhodes、Wurlitzer 或 Hohner 电钢琴的经典音效带上舞台。

MainStage 的精彩不仅在于拥有软件乐器和特效，它还满载各种创新功能，让你在舞台上大放异彩。如果是初学者，你可以设定浮动拆分点，它可以随着你的演奏智能地移动分离键盘 Patch 上的分离点。在键盘从低音开始演奏时，分离点会向上移动。这样当你演奏到较高音节时，低音就不会突然变声。

有了多项映射，你可以将多项参数与同一个控制钮对应。这样无需同时转动多个旋钮，即可流畅地控制音效。当你将和弦移至新 Patch 时，凭借无缝 Patch 切换，仍能使其保持同一个声音。

合成器中的大师，Alchemy 全新登场。

Alchemy 将史诗般壮丽的声音搬上舞台。丰富多样的预置，实时呈现的转音选项，尽在你的指尖跃动，其效果媲美出色的硬件键盘。无论是再现复古模拟合成音效，演奏超现代采样乐器，还是表现其他音乐风格，你都能找到充满灵感的音效。

进一步了解 Alchemy

吉他手，
无需乐器，即可获得各种音效。

由于有了 MainStage，你可以利用 Amp Designer、Bass Amp Designer 和 Pedalboard 中所有的美妙乐音，以及你喜欢的 Audio Units 吉他音效进行现场演奏。你还能获得全屏调谐器、脚控节拍和单块效果器图形，它们会随着每个 Patch 而变化，以表现正在使用的不同 Pedalboard。有了无缝 Patch 切换功能，在不同 Patch 间切换时，音调与和弦可以自然减弱。MainStage 还支持 Apogee GiO USB 音频接口和脚踏控制器，让你不必动一根手指，就能操作 Pedalboard、Wah、旋转扬声器音效以及回放插件。

歌手，
全面兼顾，尽可全情演出。

有了 MainStage，你在演出时就能全神贯注。你可以解放双手，使用脚踏板来控制烘托演唱的特效；在屏幕上播放对比明显的大号注释，如歌词与和音变化。这样，你只需顾及一件事：全心地演唱。

全套演出，一套即成。

喜欢录音时的声效？MainStage 3 让同样的乐器和音效轻松呈现在舞台上。你所拿手的一切，包括来自 Logic Pro 的声音资源库和 Smart Control，均已囊括在 MainStage 3 之中。你甚至可以添加预先录制的背景音乐，让演出更精彩。你还会发现优化的混音器、更易用的通道条等全新功能。Mac 作为现场演出工具的核心，让安装和拆卸都更快捷，中间的各种操作也更可靠。你还能轻松使用自己喜欢的硬件，掌控演出的每一处微妙变化。

旋钮、推子和开关，好看又好用。

MainStage 和 Smart Control 的界面专为现场演出而设计。你可在高对比度和高分辨率 (支持 Retina 显示屏) 的全屏视图中查看控制钮，即使在炫丽舞台上，它们依然十分显眼。通过设置，你能让每个控制钮与每个 Patch 中的不同参数相关联。屏幕上只会显示歌曲所需要的控制钮，你再也不必苦苦搜寻正确的旋钮。

十分简便的设置

有了 MainStage，舞台装备的设置从未如此神速。你可以首先从形形色色的演出模板中任选一种，其中许多模板都预先加载了动听的 Patch。或访问全新的 Patch 资源库寻找一款 Patch 或音效，然后将其调整到理想状态。Patch 资源库还能让你取用在 Logic Pro X 中的所有 Patch。如果需要用另一台 Mac 演出，你也不必重建一切。你只需将现场演出设置保存为一个项目，MainStage 就会把所有的必要文件纳入一个文件夹，以便随身携带。

使用你心仪的硬件

如果你已经习惯使用旋钮、按钮和推子，MainStage 也不会让你失望。它几乎可以连接世界上任何一种外置 MIDI 设备，而且界面十分简单直观。只要将实体控制项指定到屏幕上的控制项，它们就会永远保持连接 (直到你改变主意为止)。你再也不必反反复复地为每个预置去指定旋钮。当你在巡演时，如果上台前硬件突然发生故障，更换起来也十分快速简单。

MainStage 3 还提供改进模式，让固定控制器旋钮可以像高级装备上不计其数的编码器旋钮一样工作。如果你的控制表面可以接收来自软件的状态信息，MainStage 将会发送 MIDI 消息，让你的控制显示屏保持完美同步。

录制精彩演出

MainStage 可让你记录所有经过的音频信号。只要将所有音频信号传送到指定的立体声输出端口，然后选择文件格式，包括标准 AIFF、WAV 或 Apple CAF，即可录制演出。几乎任何长度的演出都不在话下。

快速而轻松地编辑音效

用 Mac 制作音效要比依赖各种外置装备容易得多。你再也不必紧盯着小小的硬件显示屏，明亮的全屏显示让你轻松调整一切。你还可以随意将自己喜欢的音效组合成一个 Patch，甚至包括软件和硬件音效。

当你需要让这些音效发挥作用的时候，MainStage 能提供的选择数不胜数。多项映射功能让一个旋钮控制多个插件参数，因此只要旋转一个旋钮，就能让声音发生戏剧性的变化。EXS 编辑器可在 MainStage 中直接打开，让即时编辑采样乐器变得更简单。Patch 列表编辑器让你能够灵活编排或跳过多个 Patch，完全按你设定好的列表来演变。

分配与映射视图让你所有的硬件和屏幕控制连接一目了然；而混音器的全新功能可让你更轻松地打开、关闭、忽略或重新排列插件，无需费力寻找修饰键。

自动采样。更多乐器，任你信手拈来。

自动采样器可快速、轻松地将你的硬件合成器变成采样乐器，让你能从自己的 Mac 上进行演奏。你的键盘装备的设置与管理也变得更加简单。

聪明、快速、高效地使用 Mac。

MainStage 充分利用 Mac 的强大动力和超凡效率，实现了大多数硬件难以企及的性能和音质。例如，EXS 采样器通过使用 Mac 中所有可用的内存，将大量数据提供给单一乐器使用。通过重复利用已加载的插件，混杂的通道条可以减轻系统负荷。MainStage 还可同时利用多个内核的处理能力。即使当你将系统推向极限，它也不会错过任何一个音符。如果你碰到问题，自动存储功能将为你提供备份，让你即刻开始演出。

自定义你的面板布局

借助成组控制，你可以将整套旋钮、按钮、推子、仪表等拖放到你的布局中。你可以由此轻松自定义所有控制钮的数量、类型、尺寸和颜色。你甚至能决定每个控制钮对每个 Patch 发挥的作用。对齐参考线功能可以将一切调整到位。拷贝和粘贴功能可让你快速复制一个布局的特性并应用到另一个布局中。你可以将布局的某些部分保存为成组控制，然后在其他布局中重复使用。也可以按任何顺序排列 Patch，并随心所欲地进行修改。由于支持 Smart Control，你的布局将会适应载入的 Patch。合成器看上去一目了然，而 Vintage B3 Patch 则会提供风琴拉杆。

控制中心

MainStage 支持 ReWire 和 MIDI 输出，这意味着演出所需的一切都已纳入 Mac 之中，令你全权掌控。每个 Patch 都是一个功能完备的混音器和音频路由器，让你在演出中随时自由调转方向。这一分钟你还在整个乐队的配合下进行极度失真的吉他演奏，下一分钟你又能利用外置合成器按照 Ultrabeat 鼓声模式创作低音循环。
