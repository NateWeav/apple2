最新版本的 iTunes 現已預載於 macOS Mojave。立即升級，盡享你喜愛的音樂、電影和 podcast。你也可以透過 iTunes 加入 Apple Music，盡情串流或下載以離線播放超過 5,000 萬首歌，完全不受廣告打擾。

下載 macOS Mojave

你可隨時下載適用於以往 macOS 版本的 iTunes 12.8，以及 Windows 版應用程式。

iTunes 12.8

硬件：

  • 配備 Intel 處理器的 Mac 電腦
  • 播放 720p HD 影片、iTunes LP 或 iTunes Extras，須配備 2.0GHz Intel Core 2 Duo 或更快速的處理器
  • 播放 1080p HD 影片，須配備 2.4GHz Intel Core 2 Duo 或更快速的處理器以及 2GB RAM
  • 1024x768 或更高的螢幕解像度；播放 iTunes LP 或 iTunes Extras 須配備 1280x800 或更高螢幕解像度
  • 使用 Apple Music、iTunes Store 及 iTunes Extras 須使用互聯網連線
  • 製作音訊、MP3 或備份 CD，須使用 Apple combo drive 或超級光碟機；部分非 Apple 品牌的 CD-RW 光碟機亦可能適用。Apple Music 歌曲目錄中的歌曲不能燒錄到 CD 上

軟件：

  • OS X 10.10.5 或更新的版本
  • 400MB 可用磁碟空間
  • Apple Music、iTunes Store 及 iTunes Match 的供應狀況視乎國家或地區而定
  • Apple Music 試用須完成登記，並只限新用戶。收費計劃將於試用期後自動更新

iTunes 適用於 Windows

硬件：

  • 配備支援 SSE2 的 1GHz Intel 或 AMD 處理器，及 512MB RAM 的 PC
  • 播放 iTunes Store 的標準解像度影片，須配備 Intel Pentium D 或更快速的處理器、512MB RAM 及兼容 DirectX 9.0 的顯示卡
  • 播放 720p HD 影片、iTunes LP 或 iTunes Extras，須配備 2.0GHz Intel Core 2 Duo 或更快速的處理器、1GB RAM 以及 Intel GMA X3000、ATI Radeon X1300、NVIDIA GeForce 6150 或更佳的顯示卡
  • 播放 1080p HD 影片，須配備 2.4GHz Intel Core 2 Duo 或更快速的處理器、2GB RAM 及 Intel GMA X4500HD、ATI Radeon HD 2400、NVIDIA GeForce 8300 GS 或更佳的顯示卡
  • 1024x768 或更高螢幕解像度；播放 iTunes LP 或 iTunes Extras 須配備 1280x800 或更高螢幕解像度
  • 16 位元聲效卡及揚聲器
  • 使用 Apple Music、iTunes Store 及 iTunes Extras 須使用互聯網連線
  • 製作音訊 CD、MP3 CD、備份 CD 或 DVD，須使用兼容 iTunes 的 CD 或 DVD 燒錄機。Apple Music 歌曲目錄中的歌曲不能燒錄到 CD 上

軟件：

  • Windows 7 或更新的版本
  • 64 位元版本的 Windows 須使用 iTunes 64 位元安裝程式
  • 400MB 可用磁碟空間
  • 此版本的 iTunes 或無法再兼容部分第三方視覺效果。如要更新兼容 iTunes 12.1 或更新的版本的視覺效果，請聯絡開發人員
  • Apple Music、iTunes Store 及 iTunes Match 的供應狀況視乎國家或地區而定
  • Apple Music 試用須完成登記，並只限新用戶。收費計劃將於試用期後自動更新
