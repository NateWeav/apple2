MainStage 3

舞台聲響，媲美錄音室效果。

將你的 Mac 變成現場演奏的操控台。以大量的 plug-in 及聲音效果，為你的鍵琴、結他或人聲演出進行大變身，為觀眾打造一生難忘的音樂盛會。
在 Mac App Store 下載 MainStage 3，只需 HK$238。

操控台出色，各種表演者同樣合適。

MainStage 適合各類型的表演者。你可以盡情使用喜愛的 Logic Pro 樂器和特效 plug-in，從而將舞台設置和拆卸工序化繁為簡。配合伴奏音軌，你的聲音要多震撼，就有多震撼。此外，你還可以使用你喜愛的一切硬件控制工具。

鍵琴手，
夢幻演出正等著你。

現在，你可以將演出提升至更高層次，超越你的真實可能。Arpeggiator 具備一系列的 note-based remote control 和靈活的 latch mode；Chord Trigger 則讓你只需單按一個音符，即觸發完整的複雜和弦；透過 Retro Synth 一系列直覺化的操控選項，你可以重新創造你所鍾情的 70 及 80 年代電子音效；你亦可以將真實自然的琴音搬上舞台，包括 Hammond B3 管風琴、Hohner Clavinet D6，以及 Fender Rhodes、Wurlitzer 和 Hohner 電子琴。

MainStage 不只是令人讚歎的軟件樂器和特效總匯，更滿載創新主意，讓你在台上光芒四射。Floating Split Point 是適合入門者使用的功能，它可以隨著你的演奏，聰明地在 split keyboard Patch 上移動 split point，當你的低音旋律開始邁向高音域，split point 會隨之移動，避免你在演繹高音時，低音旋律會突然轉成其他音調。

有了 Multimapping，你可把多項參數指派到單一控制鈕，不必一次過轉動多個旋鈕，即可流暢地控制你的音效。而順暢的 Patch 切換功能則讓你在切換至新 Patch 時保持和弦不變。

Synth 中的強者 Alchemy，正式登場。

Alchemy 為現場演出帶來媲美頂級實際鍵盤的非凡音效，多不勝數的預設組和實時變聲操控功能，盡在你的指尖。無論你要演繹的是 vintage analog synth 抑或 ultramodern 經取樣樂器的重新創作作品，都可以找到扣人心弦的音效，任何音樂演奏類型也毫不例外。

進一步了解 Alchemy

結他手幾乎不需任何樂器，就能獲得任何音效。

全賴 MainStage，你可在現場演出時融入 Amp Designer、Bass Amp Designer 和 Pedalboard 的美妙音調，以及任何你喜愛的 Audio Units 結他特效。此外，全螢幕 tuner 和腳踏式 tap tempo 已為你準備就緒，並提供 stompbox 圖像化顯示，它會隨每個 Patch 而改變，呈現你在 Pedalboard stompbox 使用方面的變化。而透過順暢的 Patch 切換功能，你可以在切換 Patch 時讓音符或和弦自然地淡出。MainStage 並支援 Apogee GiO USB 音訊介面及腳踏控制器，讓你輕鬆操控 Pedalboard、wah、旋轉揚聲器特效和 Playback plug-in，完全不必動手。

歌手
全面兼顧，就可全情演出。

MainStage 讓你專注演出。你不必動手控制你融入歌聲中的特效，只需使用腳踏板即可。你的螢幕還可顯示又大又清晰的提示，例如歌詞與和弦變換，讓你的心思，統統花在歌唱上。

完整一套，完成完美演出。

喜歡自己在錄音室錄製的音效？MainStage 3 能助你將相關的樂器與特效原原本本地搬上舞台，一切輕而易舉。你熟悉的所有 Logic Pro Sound Library 和 Smart Control 功能，現已整合至 MainStage 3。你甚至可以利用預錄伴奏音軌令表演更豐富。此外，你還能夠使用各種新功能，例如更佳的 Mixer 和更易用的 channel strip 等等。利用 Mac 作為你現場演出的核心器材，舞台設置和拆卸工序就可以更快速完成，而演出過程中的一切亦會更加可靠穩妥。你亦可選用你喜愛的硬件來輕鬆控制演出的每個細節。

旋鈕、Fader 及開關，好看好用。

MainStage 和 Smart Control 介面是專為現場演出而設計，為你提供全螢幕、清晰並支援 Retina 顯示器的高解像操控顯示，即使站在舞台遠處亦可看得清清楚楚。你亦可事先作出安排，將每個 Patch 中的參數逐個設定為不同的操控選項。螢幕只顯示你的歌曲即席所需的操控選項，你便不用再苦苦找尋正確的旋鈕。

非常簡易的設定

借助 MainStage，設置你的舞台器材變得前所未有地便捷。你可先從各式各樣的表演樣板中挑選所需，當中很多已預先載入了各種動聽的 Patch；又或在全新的 Patch Library 中物色 Patch 或音效，將之進一步調校至完美。Patch Library 還讓你取用你所有在 Logic Pro X 上的 Patch。如你需要使用另一部 Mac 來演出，也不必重新建立一切，只要將你的現場表演曲目儲存為計劃案，MainStage 就會將全部所需檔案放進一個檔案夾，讓你隨身攜帶。

使用你喜愛的硬件

MainStage 支持你對旋鈕、按鈕和 fader 的迷戀。它可連接幾乎全球任何外置 MIDI 裝置，而且介面本身設計非常簡約。只要把實體控制鈕指定到螢幕上的控制鈕，它們就永遠連接，直至你改變主意為止。從此，你不用再重重覆覆為每個預設指定旋鈕。將來，萬一你上台前才遇到硬件失靈，更換起來也快捷簡易。

MainStage 3 並具備不同的 pickup mode，令固定的控制旋鈕操作起來更像在高級器材上才有的 endless encoder 旋鈕。如你的控制面板可接收來自軟件的狀態資訊，MainStage 可發出 MIDI 訊息，讓你的操控顯示保持完美同步。

錄下你的演出

MainStage 讓你收錄任何經過的音訊訊號，只需把你的所有音訊導向至一個指定的立體聲輸出，然後選擇檔案格式如標準 AIFF、WAV，甚至是 Apple CAF，就可收錄幾乎任何長度的演出。

輕鬆快速地編輯音效

利用你的 Mac 創作音效，遠比依賴各種外置器材來得簡單。有了光亮的全螢幕顯示方式，你再也不必瞇著眼睛看那細小的硬件顯示器，令一切的微調操作變得更輕鬆。你亦可將任何喜歡的音效結合成一個 Patch，就算是軟件與硬件音效也可結合。

當要用到這些音效時，MainStage 給你帶來大量選擇。Multimapping 讓一個旋鈕控制多個 plug-in 參數，想為你的音效帶來戲劇性的轉變，轉動一個旋鈕即可。EXS editor 可直接在 MainStage 中開啟，令即興編輯 sampler 樂器變得更容易。當演奏曲目隨著演出演變，Patch List editor 讓你靈活地重新編排或跳過部分的 Patch。

Assignment & Mapping 顯示方式讓你檢視所有硬件和螢幕操控的對應連接，一目了然。Mixer 的一眾功能則讓你更易開啟、關閉、繞過或重新排列 plug-in，而不用再找 modifier key。

自動取樣功能，樂器更多，操作更易。

Auto Sampler 把你的硬件 synth 變成可在你的 Mac 上演奏的經取樣樂器，過程快速簡單。要設定和管理你的鍵琴操控台，遠比以往簡單得多。

聰明、快速、有效地使用 Mac。

MainStage 善用 Mac 的強大效能與高效率，帶來絕大多數硬件望塵莫及的表現與音色。例如 EXS sampler 就可利用你 Mac 上所有可用的記憶體，為單一樂器處理龐大的數據；Aliased channel strip 功能會重用你已載入的 plug-in，以減輕系統負荷。MainStage 可運用多核心處理效能，確保每個音符都能清晰展現，即使系統效能接近極限亦然。而萬一你遇上問題，Autosave 讓你迅即取回備份，繼續演出。

自訂你的面板

Grouped Control 功能讓你將整套旋鈕、按鈕、推桿和參數表等等拖放至面板之中，然後你就可以輕鬆自訂所有操控選項的數目、類型、大小和顏色。你甚至可以為每個 Patch 單獨設定操控選項，決定當中的操控內容。對齊參考線讓一切歸位，而 Lift 及 Stamp 功能讓你快速地複製面板的特色，然後應用到另一個面板上；你更可將面板的不同部分儲存為各項 Grouped Control，並於其他面板上重用。此外，你可按任何次序編排你一眾的 Patch，還可隨時作出變動。加上對 Smart Control 功能的支援，你的面板會自行配合載入的 Patch，Synths 會以 synth 造型出現，而 Vintage B3 Patches 則會顯示管風琴的拉桿。

控制中心

MainStage 支援 ReWire 與 MIDI 輸出，換言之，你表演所需的一切都匯集到你的 Mac 上，由你完全操控。每個 Patch 都以獨立的音訊 mixer 與 router 運作，因此你可在演出期間完全改變演繹方向。這一分鐘你可在整隊樂隊伴奏下彈奏重破音結他；下一分鐘你又可以利用外置 synth 創作低音 loop，襯托 Ultrabeat 鼓聲節拍。
