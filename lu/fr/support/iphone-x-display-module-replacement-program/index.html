Programme de remplacement du module écran de l’iPhone X pour les problèmes tactiles

Apple a constaté que, dans certains cas, l’iPhone X peut présenter des problèmes tactiles dus à la possible défaillance d’un composant du module écran. Les iPhone X concernés par ce problème peuvent présenter les symptômes suivants :

  • L’écran, ou une partie de l’écran, ne répond pas ou répond de façon intermittente au toucher.
  • L’écran réagit même s’il n’a pas été touché.

Apple ou un centre de services agréé Apple remplacera gratuitement le module écran sur les appareils éligibles.

Aucun autre modèle d’iPhone n’est éligible à ce programme.

Processus de réparation

Choisissez l’une des options ci-dessous pour la réparation de votre iPhone X. Votre iPhone sera examiné avant toute intervention pour vérifier qu’il est éligible à ce programme.

  • Trouvez un centre de services agréé Apple.
  • Contactez l’assistance Apple pour organiser une réparation par envoi postal via le centre de réparation Apple.

Avant de l’envoyer en réparation, sauvegardez les données de votre iPhone via iTunes ou iCloud. Découvrez comment préparer votre iPhone en vue de sa réparation.

Si votre iPhone X a subi des dommages qui compromettent la réparation, comme un écran fissuré, ceux-ci devront être réparés en premier lieu. Dans certains cas, la réparation supplémentaire peut vous être facturée.

Informations supplémentaires

Il est possible qu’Apple réduise ou limite le service de réparation au pays ou à la région d’achat d’origine du produit. Pour les iPhone X achetés dans des pays membres de l’Espace économique européen (EEE), la réparation est possible dans d’autres pays membres de l’EEE.

Ce programme Apple n’étend pas la durée de la garantie limitée de l’iPhone X.

Si vous pensez que votre iPhone X est concerné par ce problème et si vous avez réglé des frais pour le remplacement de votre module écran, vous pouvez contacter Apple afin d’obtenir un remboursement.

Le programme couvre les iPhone X concernés durant les trois années suivant la première vente au détail du produit.

Informations en date du 09/11/2018
