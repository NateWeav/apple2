iTunes
วันนี้มาพร้อมกับ macOS Mojave

iTunes เวอร์ชั่นล่าสุด ติดตั้งมาพร้อมกับ macOS Mojave อัพเกรดวันนี้เพื่อเข้าถึงเพลง
ภาพยนตร์ รายการทีวี และพ็อดคาสท์โปรดของคุณ นอกจากนี้ iTunes ยังเป็นที่ที่คุณสามารถ
สมัครใช้ Apple Music และสตรีมหรือดาวน์โหลดเพลงกว่า 50 ล้านเพลงไว้ฟัง
แบบออฟไลน์ได้โดยไม่มีโฆษณาอีกด้วย

ดาวน์โหลด macOS Mojave

คุณสามารถดาวน์โหลด iTunes 12.8
สำหรับ macOS เวอร์ชั่นก่อนหน้าได้เสมอ
เช่นเดียวกับแอพพลิเคชั่นสำหรับ Windows

iTunes 12.8

ฮาร์ดแวร์:

  • คอมพิวเตอร์ Mac ที่มีโปรเซสเซอร์ Intel
  • ในการเล่นวิดีโอความละเอียด HD 720p, iTunes LP หรือ iTunes Extras ต้องใช้โปรเซสเซอร์
    Intel Core 2 Duo ความเร็ว 2.0GHz หรือเร็วกว่า
  • ในการเล่นวิดีโอความละเอียด HD 1080p ต้องใช้โปรเซสเซอร์ Intel Core 2 Duo ความเร็ว
    2.4GHz หรือเร็วกว่า และ RAM ขนาด 2GB
  • ต้องมีความละเอียดหน้าจอ 1024 x 768 หรือสูงกว่า
    และความละเอียด 1280 x 800 หรือ
    สูงกว่า สำหรับการเล่น iTunes LP หรือ iTunes Extras
  • การเชื่อมต่ออินเทอร์เน็ตเพื่อใช้ Apple Music, iTunes Store และ iTunes Extras
  • การใช้ Apple Combo Drive หรือ SuperDrive เพื่อสร้างซีดีเสียง, ซีดี MP3 หรือซีดี
    สำรองข้อมูล ในบางครั้งอาจสามารถใช้เครื่องบันทึก CD-RW บางเครื่องที่ไม่ใช่ของ Apple
    ได้เช่นกัน ทั้งนี้เพลงจากแค็ตตาล็อกของ Apple Music ไม่สามารถนำไปเบิร์นหรือสร้าง
    เป็นซีดีได้

ซอฟต์แวร์:

  • OS X เวอร์ชั่น 10.10.5 หรือใหม่กว่า
  • พื้นที่ว่างบนดิสก์ 400MB
  • ความพร้อมในการให้บริการ Apple Music, iTunes Store และ iTunes Match อาจ
    แตกต่างกันไปในแต่ละประเทศ
  • ต้องสมัครเพื่อทดลองใช้บริการ Apple Music และเปิดให้เฉพาะสมาชิกใหม่เท่านั้น
    หลังหมดช่วงทดลองใช้ แผนบริการจะมี
    การต่ออายุโดยอัตโนมัติ

iTunes สำหรับ Windows

ฮาร์ดแวร์:

  • PC ที่มีโปรเซสเซอร์ Intel หรือ AMD ความเร็ว 1GHz ที่รองรับ SSE2 และ RAM
    ขนาด 512MB
  • ในการเล่นวิดีโอความละเอียดมาตรฐานจาก iTunes Store
    ต้องใช้โปรเซสเซอร์
    Intel Pentium D หรือเร็วกว่า, RAM ขนาด 512MB และการ์ดวิดีโอที่รองรับ DirectX 9.0
  • ในการเล่นวิดีโอความละเอียด HD 720p, iTunes LP หรือ iTunes Extras ต้องใช้โปรเซสเซอร์
    Intel Core 2 Duo ความเร็ว 2.0GHz หรือเร็วกว่า, RAM ขนาด 1GB และกราฟิกโปรเซสเซอร์
    Intel GMA X3000, ATI Radeon X1300 หรือ NVIDIA GeForce 6150 หรือดีกว่า
  • ในการเล่นวิดีโอความละเอียด HD 1080p ต้องใช้โปรเซสเซอร์ Intel Core 2 Duo ความเร็ว
    2.4GHz หรือเร็วกว่า, RAM ขนาด 2GB และกราฟิกโปรเซสเซอร์ Intel GMA X4500HD,
    ATI Radeon HD 2400 หรือ NVIDIA GeForce 8300 GS หรือดีกว่า
  • ต้องมีความละเอียดหน้าจอ 1024 x 768 หรือสูงกว่า
    และความละเอียด 1280 x 800 หรือ
    สูงกว่า สำหรับการเล่น iTunes LP หรือ iTunes Extras
  • การ์ดเสียง 16 บิต และลำโพง
  • การเชื่อมต่ออินเทอร์เน็ตเพื่อใช้ Apple Music, iTunes Store
    และ iTunes Extras
  • เครื่องบันทึกซีดีหรือดีวีดีที่สามารถทำงานร่วมกับ iTunes สำหรับสร้างซีดีเสียง, ซีดี MP3
    หรือซีดีและดีวีดีสำรองข้อมูล ทั้งนี้เพลงจากแค็ตตาล็อกของ Apple Music ไม่สามารถนำไป
    เบิร์นหรือสร้างเป็นซีดีได้

ซอฟต์แวร์:

  • Windows 7 หรือใหม่กว่า
  • Windows รุ่น 64 บิต จำเป็นต้องใช้ตัวติดตั้งแบบ 64 บิต
    สำหรับ iTunes
  • พื้นที่ว่างบนดิสก์ 400MB
  • วิชวลไลเซอร์จากบริษัทอื่นอาจไม่สามารถทำงานร่วมกับ iTunes รุ่นนี้ได้ โปรดติดต่อผู้พัฒนา
    เพื่อขอรับวิชวลไลเซอร์รุ่นอัพเดทที่สามารถทำงานร่วมกับ iTunes 12.1 หรือใหม่กว่าได้
  • ความพร้อมในการให้บริการ Apple Music, iTunes Store และ iTunes Match อาจ
    แตกต่างกันไปในแต่ละประเทศ
  • ต้องสมัครเพื่อทดลองใช้บริการ Apple Music และเปิดให้เฉพาะสมาชิกใหม่เท่านั้น
    หลังหมดช่วงทดลองใช้ แผนบริการจะมี
