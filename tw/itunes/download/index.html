下載 macOS Mojave

你可隨時下載適用於較早版本 macOS 的 iTunes 12.8，及適用於 Windows 的應用程式。

iTunes 12.8

硬體：

  • 配備 Intel 處理器的 Mac 電腦
  • 播放 720p HD 影片、iTunes LP 或 iTunes Extras，需要 2.0GHz Intel Core 2 Duo 或更快的處理器
  • 播放 1080p HD 影片需要 2.4GHz Intel Core 2 Duo 或更快的處理器，以及 2GB RAM
  • 需配備 1024x768 或更高的螢幕解析度；播放 iTunes LP 或 iTunes Extras 需要 1280x800 或更高的螢幕解析度
  • 使用 Apple Music、iTunes Store 及 iTunes Extras 需要 internet 連線
  • 可使用 Apple 複合式光碟機或超能光碟機以製作音訊、MP3 或備份 CD；部分非 Apple 出品的 CD-RW 燒錄機亦可能適用 。來自 Apple Music 目錄中的歌曲不得燒錄為 CD。

軟體：

  • OS X 版本 10.10.5 或後續版本
  • 400MB 可用磁碟空間
  • Apple Music、iTunes Store、iTunes Match 的適用情況可能視個別國家或地區而異
  • Apple Music 試用須註冊，並僅適用於新訂戶。訂閱方案將在試用結束後自動續訂。

適用於 Windows 的 iTunes

硬體：

  • 配備支援 SSE2 的 1GHz Intel 或 AMD 處理器，以及 512MB 記憶體的 PC 電腦
  • 播放來自 iTunes Store 的標準畫質影片，需要 Intel Pentium D 或更快的處理器、512MB RAM，以及與 DirectX 9.0 相容的顯示卡。
  • 播放 720p HD 影片、iTunes LP 或 iTunes Extras，需要 2.0GHz Intel Core 2 Duo 或更快的處理器、1GB 記憶體，以及 Intel GMA X3000、ATI Radeon X1300 或 NVIDIA GeForce 6150 或更高階的繪圖晶片。
  • 播放 1080p HD 影片需要 2.4GHz Intel Core 2 Duo 或更快的處理器、2GB RAM，以及 Intel GMA X4500HD、ATI Radeon HD 2400，或是 NVIDIA GeForce 8300 GS 或更高階的繪圖晶片。
  • 需配備 1024x768 或更高的螢幕解析度；播放 iTunes LP 或 iTunes Extras 需要 1280x800 或更高的螢幕解析度。
  • 16 位元音效卡與揚聲器
  • 使用 Apple Music、iTunes Store 及 iTunes Extras 需要 internet 連線
  • 可使用與 iTunes 相容的 CD 或 DVD 燒錄機以製作音訊 CD、MP3 CD、備份 CD 或 DVD。來自 Apple Music 目錄中的歌曲不得燒錄為 CD。

軟體：

  • Windows 7 或後續版本
  • 64 位元版本的 Windows 須使用 iTunes 64 位元安裝程式
  • 400MB 可用磁碟空間
  • 部分協力廠商的視覺效果模組可能與此版本的 iTunes 不相容。請聯繫開發者，取得與 iTunes 12.1 或後續版本相容的更新版視覺效果模組
  • Apple Music、iTunes Store、iTunes Match 的適用情況可能視個別國家或地區而異
  • Apple Music 試用須註冊，並僅適用於新訂戶。訂閱方案將在試用結束後自動續訂。
