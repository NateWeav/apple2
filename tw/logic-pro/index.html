進一步了解外掛模組和音效

Patch 的強大威力。

Sound Library 內含豐富的 Patch，讓你能快速載入複雜的多聲道音效。而 Smart Control 能簡化微調與編輯的過程，讓你得到想要的音質。

眾多循環樂段，快速打造好聲音。

內建超過 7,000 種免版稅的 Apple Loop，已標記關鍵字，且能配合你的計畫案自動調和拍速與音調。你可從中選擇，快速譜曲。你也可以使用其他來源、未經標記的循環樂段，他們同樣會依照你計畫案的節拍自動播放。

全新

Smart Tempo。

讓眾音軌合拍。

Smart Tempo 可自動管理整個計畫案內容的節拍。自由錄製，不需使用節拍音軌。從黑膠唱片樣本到現場樂器演奏等音訊，可輕鬆合併和剪輯，不必受限於原有的節拍、速度變化或來源。

Flex Time。 控制時間點， 一點不費時。

使用 Flex Time，快速控制錄音的節拍時間點與節奏。輕鬆地在波形表上移動個別節拍，以修正鼓聲、人聲、吉他，或是其他種類的音軌，完全不必切割和移動區段。

Flex Pitch。音準，精準掌握。

使用 Flex Pitch 修改個別音符的音量與音調，輕鬆快速。只要移動到任一音符上，就會出現所有參數選項以供調整。

Logic Remote。

你的音樂，你一手掌握。

Logic Remote 可將你的 iOS 裝置無線連接至 Mac，讓你使用 Multi-Touch 的功能來控制演出與混音。從你的 iPad Pro、iPad 或 iPhone，彈奏 Logic Pro X 上任何的軟體樂器，並隨時輕鬆調整你的音色。無論在電腦旁或是房間另一頭，你總是能在最舒適愜意的地方進行作業。

混音，用 Multi-Touch。

你可以在房間任一處操控混音，以 Multi-Touch 推子找出最佳音質。

配個對，玩音樂。

各種可在螢幕上彈奏的樂器，例如鍵盤、吉他與鼓組板等，讓你在 iPad 上彈奏 Logic Pro X 中的軟體樂器。

Drummer。

何止加鼓聲音軌，簡直像加進新團員。

使用 Drummer 就如同請了一位伴奏鼓手，或與技巧高超的節拍程式設計師搭擋合作。利用 Drummer 的智慧技術，製作原汁原味的原聲鼓聲音軌或電子節拍。眾多鼓手專擅多樣類型音樂，可供你選擇，且使用簡單的控制方式就能指揮演奏。

建立個人的原聲鼓或電子鼓組。

你可以使用為原聲鼓組而設的 Drum Kit Designer，和為電子鼓聲音軌而設的 Drum Machine Designer 來自訂音色。Drummer 原有各自的設定，但你也可以換掉個別鼓件並修飾音色，以符合你所設想的演奏效果。

作曲和演出。

功能齊全的錄音室，就在你 Mac。

Logic Pro X 可讓你的 Mac 化身為掌控繁複計畫案的專業級錄音室。你可以錄下作曲與演出，例如現場樂團與單人軟體樂器演奏，並將這些樂音納入你的歌曲。

終極錄音方式。

流暢的播放中即時錄音、自動音樂片段管理、支援原始 24 位元/192kHz 音訊，Logic Pro X 讓一切做起來輕鬆寫意，還原重來也一樣容易。擁有超過 250 道可用音軌，可執行數百個外掛模組，完成計畫案所需的一切，你已全盤掌握。

MIDI，玩得出神入化。

遠遠勝過一般的音序器，Logic Pro X 擁有一整套先進的選項功能，讓你進行錄音、剪輯與操控 MIDI 演奏。以基於區段的參數設定，調整音符速度、時間點、動態等設定，將原本鬆散的演奏化為節奏緊湊的動聽作品。你也可以使用 Smart Quantize，讓 MIDI 演奏更加緊湊的同時，還保有如 flam、chord roll 等音樂細節。

先進工具，引領業界。

在你製作歌曲的過程中，Logic Pro X 可助你整理所有靈感，並選出最完美的內容。將相關音軌分組、試聽不同版本，並整合多道音軌。超快速的點按拖移合成功能，讓你在眾多音樂片段中構築最佳演出。

Track Alternative

你可以為某道音軌或多個群組音軌製作備選版本，並隨時在其間切換，試聽不同的版本選項。為某一音軌區段建立、儲存並挑選不同的剪輯與編曲，以更輕鬆的方式試驗各種創意構想。

Take 和 Quick Swipe Comping

只須按住並拖移，即可選擇每個音樂片段的最佳區段，創作完美無瑕的合成，完成過場流暢的交叉漸變。你可儲存多段合成並在其中來回切換，挑選你最喜愛的。

Track Stack

將多道相關音軌統整為單一音軌；使用 Summing Stack 快速製作副混音；或製作分層且可分拆的樂器。

Project Alternative

你可依需要建立多個計畫案備選版本，每個版本都有自己的名稱與設定，但共用相同素材，有效節省儲存空間。在你載入任一版本加以改動時，完全不會影響原始版本。

Track Group 與 VCA Fader

使用 Track Group 與 VCA 推子來管理大型混音。可將任一聲道指派至任一音軌群組，然後在群組內的任何單一聲道裡，就能控制群組內所有音軌的音量或其他參數。

Automation

輕鬆記憶任何聲道控制排或外掛模組參數的變動。只要啟用自動混音，按下 Play，再進行更改即可。

更多專業功能，展現專業混音。

Logic Pro X 滿載各種絕佳工具與資源，即使你已是熟練專家，也能讓技術更臻完美，使創意與工作流程更上層樓。
