Logic Pro X

制作的动力十足， 创作的十足强音。

新功能

插件和音效。

包罗万象的音效与乐器。

可快速取用一系列插件和音效，让你尽情挥洒创意。

进一步了解插件和音效

Patch 的威力。

声音资源库包含海量的 Patch，让你能快速载入复杂的多声道音效。智能控制功能则可以帮你简化获得合适音效的调整和编辑过程。

从乐段起步，旋律跃然而出。

内置有超过 7000 款免版税 Apple 乐段供你选择。这些乐段均带有关键词标签，并能自动与项目的速度和音调保持一致，从而让你能快速创作曲目。你甚至可以使用其他来源的无标签乐段，它们将根据项目的节奏自动播放。

新功能

有了智能速度，

条条音轨都合拍。

智能速度可自动管理项目内所有内容的节拍，无需使用节拍音轨，就能自由录制。包括黑胶唱片小样、现场乐器演奏在内的各种音频，无论其原始节拍如何，速度怎样变化，甚至不管什么来源，都能轻松地组合并编辑。

有了 Flex Time，控制时间点，一点不费时。

使用 Flex Time 可快速操控录音成品的时间点和节奏。无需分切和移动片段，只要在波形图上调整个别节拍，就能轻松修正鼓声、人声、吉他或其余任何种类的音轨。

Flex Pitch 工具，让音准变得更准。

使用 Flex Pitch 可轻松快捷地编辑每个音符的音量和音高。滑动任一音符，所有的参数就会出现供你调整。

Logic Remote 在手，

音乐创作尽在掌握。

Logic Remote 可将 iOS 设备与 Mac 无线连接，让你能利用多点触控的优势来控制演奏和混音。你可以在 iPad Pro、iPad 或 iPhone 上演奏 Logic Pro X 中的任意软件乐器，并随时随地轻松塑造音形。无论是近在电脑旁，还是远在房间另一头，每个舒适自在的角落都是你进行创作的好地方。

多点触控，混音多一手。

你能在房间内的任何位置使用多点触控推子控制混音，以获得绝佳的音效。

配起来，奏起来。

在 iPad 上使用键盘、吉他和鼓垫等众多屏幕乐器，演奏 Logic Pro X 中的各种软件乐器。

Drummer，

加的不只是条鼓点音轨，更是个乐队搭档。

使用 Drummer 就如同雇了一位伴奏鼓手，或是在与高水平的节拍编排者合作。利用 Drummer 的智能技术，可创建逼真的原声鼓音轨或电子节拍。你可从数十位不同音乐流派的鼓手中进行选择，并使用简单的控制功能来指挥演奏。

你可以使用 Drum Kit Designer 来定制原声架子鼓，用 Drum Machine Designer 来定制电子鼓音轨。鼓手们带来了他们的设置搭配，但是你也能替换个别鼓件或修整声效，以符合你所设想的效果。

从作曲，到演奏，

整个音棚都在 Mac 里。

Logic Pro X 让你的 Mac 化身专业录音室，即使处理要求极高的项目也游刃有余。无论是乐队现场演出或软件乐器独奏，你都能轻松记录下你的创作和演奏，并将它们融入你的乐曲中。

理想的录音方式。

流畅的穿插录音、自动片段管理、原始 24-bit/192kHz 音频支持，在 Logic Pro X 中，这些全都易于执行，也便于撤销。250 多个可用音轨和数百种插件的运行能力，让你能得心应手地完成项目。

MIDI，玩得出神入化。

Logic Pro X 超越一般音序器，拥有一整套进阶选项，便于你录音、编辑以及控制 MIDI 演奏。使用音符力度、时序、动态等基于片段的参数，将零散的演奏结合成一组严谨的音乐律动。借助智能量化功能，你也可以让 MIDI 演奏更为紧凑，同时还保留装饰音或分解和弦等音乐细节。

出类拔萃的音乐制作工具。

创作乐曲的过程中，你可利用 Logic Pro X 整理所有创意灵感并严加甄选。你可以将相关音轨分组，试听备选版本，或合并多条轨道。快捷的点按并拖移伴奏功能，帮你利用多个片段制作最优演奏。

备选轨道

不论是单条音轨，或成组音轨，你都可以为之创建多个备选版本，并随时切换，试听不同的选项。你可以为音轨中的片段创建、存储、选择不同的剪辑版本或编曲方式，从而更轻松地尝试各种创意构想。

片段与快速扫动伴奏

只需选中每个片段的最佳段落，然后点按并拖移，就能创建无缝衔接的伴奏，并加入流畅过渡的交叉渐变。你可以保存多个伴奏版本，切换试听以选出你最满意的一个。

Track Stack

你能将多个相关轨道整合为一个。使用总和堆栈，可以快速创建子混音，也能创建多层的或分离的乐器音轨。

项目备选

你可以随心所欲地为一个项目创建多个备选版本，每个版本均可独立命名并进行设置，但它们共享相同的素材资源，这样能有效节省存储空间。载入任何版本都可进行更改，又不会影响原始版本。

轨道组和 VCA 推子

使用轨道组和 VCA 推子可管理大型混音项目。将任何选定的通道分配到一个轨道组，然后通过该组任意一条通道，便可控制该组所有轨道的电平或其他参数。

自动化

可轻松记录对任何通道条或插件参数所做的更改。只需启用自动化功能，按下“播放”，即可进行更改。

更多专业混音工具，应有尽有。

Logic Pro X 满载令人惊叹的工具和资源，在你精雕细琢自己的作品时，能帮你增强创造力并优化工作流程，即使是专业高手也能深受其益。
