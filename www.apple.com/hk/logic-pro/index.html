Plug-in 及聲音效果。

音效及樂器陣容強大。

即時取用一系列 plug-in 和聲音效果，讓你盡情揮灑創意。

進一步了解 plug-in 及聲音效果

一系列 Patch，威力強大。

Sound Library 提供大量的 Patch，讓你可以快速載入複雜的多聲道音效。而 Smart Control 則可簡化調校聲音與編輯的過程，讓你更快得到想要的音效。

取用眾多 Loop 即時作樂。

選用內置的過 7000 款免版稅並已標記關鍵字的 Apple Loop，可自動配合專案的速度與音調，快速製作你的樂曲。你也可使用其他來源提供的未標記 loop，還能配合你專案的速度自動播放。

新登場

Smart Tempo。

眾多音軌從此合拍。

Smart Tempo 可自動管理專案中所有內容的速度，讓你無需既定音軌就可自由錄製內容。輕鬆結合並編輯音訊，不受原始速度、速度變化或甚至來源所限，無論是黑膠碟 sample 還是現場樂器演奏都可輕易處理。

Flex Time，
時間控制，更省時間。

運用 Flex Time 來快速操控錄音的定時與速度。輕鬆在波形表上移動個別節拍以修正鼓聲、人聲、結他或其他種類的音軌，無需再切割與移動區段。

Flex Pitch，
音高，至臻完美。

以 Flex Pitch 快速編輯個別音符的音量與音高。移動到任何音符上，即會呈現所有參數以供修改。

Logic Remote。

音樂隨你掌控。

Logic Remote 可將你的 iOS 裝置無線連接至你的 Mac，讓你盡情運用 Multi-Touch 的強大本領來控制演奏及混音。利用你的 iPad Pro、iPad 或 iPhone 演奏 Logic Pro X 的任何軟件樂器，更可隨時隨地輕鬆塑造聲音效果。不論是在電腦旁或是房間的另一側，你隨時都可在最方便就手的地方掌控你的音樂。

Multi-Touch 混音，由你指點。

在房間任何位置都可透過 Multi-Touch fader 來控制混音，找到最完美的音效。

一配即對，即玩音樂。

使用螢幕上的各種樂器，如鍵琴、結他及鼓組 pad，在你的 iPad 上演奏 Logic Pro X 的任何軟件樂器。

Drummer。

豈止加入鼓聲，更是加入鼓手。

運用 Drummer，好像聘用了一位鼓手或與技術高超的節拍程式員合作。Drummer 的智能技術讓你創作充滿生氣的原音鼓樂音軌或電子節拍。數十個以不同音樂風格的 drummer 任你選擇，透過簡易的控制功能就可指揮表現。

建立個人的原音或電子鼓組。

利用為原音鼓組而設的 Drum Kit Designer 和為電子鼓樂音軌而設的 Drum Machine Designer，自訂你的聲音效果。Drummer 有各自的設定，但你也可以替換個別的鼓聲和調整音效，以配合你理想中的效果。

作曲及演奏。

設備齊全的錄音室，就在你 Mac 上。

Logic Pro X 讓你的 Mac 化身專業錄音室，即使是最複雜的專案都遊刃有餘。記錄你的作曲與演出，不論錄製樂隊現場演出還是單人演奏軟件樂器皆可從容應對，讓你的出色創作盡收歌曲之中。

終極錄音方式。

流暢的 punch recording，自動化的 take 管理功能，支援毫不失真的 24 位元/192kHz 音訊。Logic Pro X 讓一切都輕鬆易用，亦更容易修正還原。備有超過 250 條音軌，可以執行數百個 plug-in，讓你隨時齊備完成專案的一切所需。

MIDI，玩得出神入化。

Logic Pro X 不僅是一款普通的 sequencer，它配備一系列先進選項，可讓你錄製、剪輯及修改 MIDI 演奏效果。使用以區段作基礎的參數來調節音符速度、定時及力度變化，將鬆散的演奏轉化成節奏緊湊的美妙音色。或潤飾你的 MIDI 演奏，同時以 Smart Quantize 保留 flam 或 chord roll 等音樂細節。

領先業界製作工具。

Logic Pro X 可隨著你創作歌曲，助你整理各種靈感並選出最合適的內容。你可將音軌組合成群組，試聽不同版本，並整合多個音軌。快如閃電的點按及拖放式合成操作，助你從多個 take 中製成你最出色的大作。

Track Alternative

製作不同版本的音軌或多個加入群組的音軌，並隨時在這些音軌之間來回切換，以試聽各個版本的效果。你可在音軌區域中製作、儲存和選取各區段的不同剪輯版本和編曲，藉此更輕鬆地試驗種種創意構思，細聽當中變化。

Take 與 Quick Swipe Comping

點按與拖放每一 take 的最佳區段，即可製作出完美的合成片段，並加入多款過場流暢的 crossfade。儲存多個合成片段，並在當中切換比較，以挑選最合你心意的一個。

Track Stack

將多個相關的音軌整合至單一音軌。使用 Summing Stack 來快速製作 submix，或製作分層式及分拆式樂器。

Project Alternative

隨心所欲為專案製作不同版本，數量不限。每個專案均可自訂名稱及設定，但同時共用相同的素材，有效為你節省儲存空間。載入任何版本來作出修改，亦不會影響原始內容。

Track Group 及 VCA Fader

運用 Track Group 及 FCA fader 來管理大型混音創作。指定任何所選聲道至 track group，然後只需對群組內某一聲道操作，即可操控群組內所有音軌的音量或其他參數。

自動化功能

輕鬆地記錄任何 channel strip 或 plug-in 參數的改變，只要啟動自動化功能，按下播放，然後進行更改即可。

更多專業功能，一應俱全。

Logic Pro X 滿載各種出色工具及資源，助你在創作過程中提升創意及工作流程，即使你已經是專業老手也可從中受惠。
