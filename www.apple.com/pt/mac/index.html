macOS

Todos os Mac utilizam o sistema operativo macOS. Permite fazer o que pura e simplesmente não consegue com outros computadores, porque o macOS foi concebido especificamente para aproveitar toda a capacidade do hardware. E vice‑versa.

O Final Cut Pro foi criado para os profissionais de vídeo mais criativos, porque dispõe de funções avançadas de edição, organização de conteúdos multimédia e um desempenho especificamente otimizado, tanto para os computadores Mac como para o macOS Mojave.

Saiba mais sobre o Final Cut Pro X

O Logic Pro é um autêntico estúdio de gravação e produção MIDI para o seu Mac, com tudo o que precisa para compor, gravar, editar e misturar áudio de uma forma inovadora. Seja qual for o tipo de música, com a enorme coleção de plug-ins e milhares de sons e loops, não faltam ferramentas para tocar as primeiras notas e criar o seu maior êxito.
