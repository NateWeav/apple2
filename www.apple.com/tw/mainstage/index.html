MainStage 3

舞台效果，媲美錄音室聲效。

將你的 Mac 變成現場演出設備。透過龐大的 plug-in 和音效，你的鍵盤、吉他或歌唱表演從此讓人改觀，這場演出也將令觀眾難以忘懷。
自 Mac App Store 下載 MainStage 3 只要 NT$890

絕佳操控台，每個表演者都好用。

MainStage 適合各類型表演者。你可使用 Logic Pro 裡所有喜愛的樂器及音效 plug-in，簡化設定與拆卸過程。透過背景伴奏音軌，達到你想要的震撼效果，並使用任何你喜愛的硬體控制器。

鍵盤手，
夢幻演出蓄勢待發。

現在，你可讓你的演出神乎其技。Arpeggiator 具備以音符為基礎的遠端控制功能與靈活的 latch 模式。Chord Trigger 則讓你只需按下一個音符，即可觸發完整的複雜和弦。透過 Retro Synth，你可使用一套直覺的 control 項目，重新營造出喜愛的 70 及 80 年代電子音色，或是在舞台上呈現 Hammond B3 風琴、Hohner Clavinet D6 或 Fender Rhodes、Wurlitzer 與 Hohner 電子琴的真實原音。

MainStage 不止是出神入化的軟體樂器與音效主機，更充滿各種創新，讓你在舞台上發光發熱。入門者可以使用 Floating Split Point，它會以智慧方式移動分離鍵盤 Patch 的 split point，以回應你彈奏的樂曲。當你用鍵盤開始彈奏低音，split point 會向上移動，因此當你進入較高音時，低音就不會突然變聲。

透過 Multimapping 功能，你可為單一控制項目對應多個參數，因此你可以順暢地操控聲音，而不必一次試著轉動好幾個旋鈕。順暢的 Patch 切換，可讓你將和弦移至新的 Patch 時，仍保持同一種音色。

Alchemy 登場，合成器中的強者。

Alchemy 為舞台帶來磅礡音色，效果媲美最出色的硬體音樂鍵盤。龐大的預設組選集與即時音色轉換的控制項目，全都任你指間掌控。無論你想再現懷舊的類比合成器，或是超現代的取樣樂器，不管你想表現何種音樂，都能找到賦予你靈感的種種音色。

進一步了解 Alchemy

吉他手，無需任何配備，即可獲得各種音效。

有了 MainStage，你可在現場演出時彈奏 Amp Designer、Bass Amp Designer 與 Pedalboard 裡所有美妙音調，並加上任何你喜愛的 Audio Unit 吉他音效。你還可擁有全螢幕 tuner、腳踏式控制 tap tempo，以及隨每個 Patch 改變的 stompbox 圖像，呈現你在不同 Pedalboard stompbox 的使用變化。順暢的 Patch 切換讓你轉換 Patch 時，音調及和弦會自然逐漸減弱。MainStage 支援 Apogee GiO USB 音訊介面及腳踏控制器，讓你不必動手即可操作 Pedalboard、wah、迴旋揚聲器效果與 Playback plug-in 等。

歌手，輕鬆全面兼顧，盡可全情演出。

MainStage 讓你專注於你的演出，不必動手，即可使用腳踏板來控制你加入歌聲裡的特效，並可在你的螢幕上顯示大型、高對比的備忘錄，如歌詞及和弦的變化，因此，你只需全神貫注地唱歌。

全套演出，一套即成。

喜歡你在錄音室的作品嗎？MainStage 3 讓你輕鬆將同樣的樂器與音效帶上舞台。你所熟悉 Logic Pro 的一切，無論是 Sound Library 或是 Smart Control 等皆已整合在 MainStage 3 中。你甚至可以藉著預錄背景伴奏音軌，讓表演更精彩。你還能找到各種新功能，如改良的 Mixer、更易於使用的 channel strip 等。將 Mac 當作現場演出設備的核心，設定與拆卸都更為快速，而中間的一切過程也變得更可靠，你可輕鬆使用最愛的硬體控制演出各項細節。

旋鈕、推桿及開關，好看好用。

MainStage 與 Smart Control 介面是特別為現場演出而設計，你擁有全螢幕、高對比及高解析度 (支援 Retina 顯示器) 顯示的控制介面，即使在舞台上也格外顯眼。你還可以進行設定，讓每個 Patch 的參數連結到不同的 control 項目，只有歌曲需要的 control 項目會顯示於螢幕上，你永遠不必費心尋找正確的旋鈕。

極簡單的設定

有了 MainStage，設定你的舞台設備，速度更勝以往。首先可從一系列完整的演奏樣板中選擇，許多樣板都預先載入了音色優美的 Patch；或是查看全新的 Patch Library，尋找用來調整至完美的 Patch 或音效。Patch Library 還可讓你存取 Logic Pro X 裡所有的 Patch。如果你需要使用另一部 Mac 演出，也不必重建一切，只要將你的現場表演組儲存為一個計畫案，MainStage 就會將所有必要檔案放在單一檔案夾之中，讓你隨身攜帶。

使用你喜愛的硬體

MainStage 支援你對旋鈕、按鈕及推桿的熱愛，它能連接世界上任何一台外接 MIDI 裝置，而介面本身也相當簡潔。只要將實體 control 項目指定至螢幕 control 項目，就會永遠保持連結 (直到你改變主意)，不必為每個預設組一再指定旋鈕。當你在巡迴演出時，如果上台前突然硬體故障，也可快速輕鬆地替換。

MainStage 3 也提供拾音器模式，使固定的控制器旋鈕，操作起來更像高階設備的無段式編碼器旋鈕。如果你的控制介面可接受來自軟體的狀態資訊，MainStage 便會傳送 MIDI 訊息，使你的控制顯示器保持完美同步。

錄下演出的精彩片刻

MainStage 讓你錄下所有通過的音訊訊號。只要將所有音訊導向指定的立體聲輸出，然後選擇檔案格式 (包括標準的 AIFF、WAV 或甚至 Apple CAF)，即可錄製各種時間長度的演出。

快速輕鬆地編輯音效

透過 Mac 創造音效遠比仰仗各種外接設備更加簡單。你擁有明亮的全螢幕檢視畫面，使你可以輕鬆調整一切，而非瞇眼看著極小的硬體顯示器。你也可結合任何喜歡的音效，即使是將軟體與硬體音色結合在同一個 Patch 上也沒問題。

當你需要運用這些音效時，MainStage 將提供你眾多選擇。Multimapping 功能允許一個旋鈕控制多個 plug-in 參數，只要轉動一個旋鈕，即可大幅改變你的音效。EXS editor 可在 MainStage 中直接開啟，讓即時編輯 sampler 樂器更加輕鬆。而 Patch List editor 可讓你隨著設定列表的演變，靈活地重新編排或跳過部分的 Patch。

Assignment 與 Mapping 檢視畫面讓所有硬體與螢幕控制項目的連結一目瞭然。而 Mixer 中的功能則可更輕鬆地開啟、關閉、略過或重新排列 plug-in，無須借助 modifier key。

自動取樣。更多樂器，輕鬆演奏。

Auto Sampler 可快速且輕鬆地將你的硬體 synth 轉變成取樣樂器，使你能自 Mac 彈奏。你的鍵盤設備設定及管理，變得更加簡單。

聰明、快速、有效率地使用 Mac

MainStage 充分利用 Mac 的威力與效率，實現絕大多數硬體不能達到的效能與音質。例如，EXS sampler 可透過使用 Mac 所有可用的記憶體，讓大量數據供單一樂器使用。透過重複使用已載入的 plug-in，滑順處理 channel strip 以減輕系統負荷。MainStage 能使用多個核心的處理威力，確保每個音符都能聽見，甚至你把系統推至極限也沒問題。如果你遇到問題，Autosave 功能可讓你獲得備份並立刻演出。

自訂你的版面

Grouped Control 讓你將整組旋鈕、按鈕、推桿、音量表等拖放至版面中，然後你可以輕鬆自訂所有 control 項目的數目、類型、大小和顏色，你甚至可以決定每個 control 項目在每個 Patch 負責的部分。對齊參考線讓一切各安其位。Lift 與 Stamp 功能讓你快速將一個版面的特色複製至另一版面。你可將部份版面儲存為 Grouped Control，並在其他版面重複使用，還可將 Patch 以任何順序編排，隨心所欲加以變動。透過支援 Smart Control，你的版面會隨載入的 Patch 而變化。Synth 的外觀會以 synth 造型出現，而 Vintage B3 Patch 則會顯示風琴音栓。

控制中心

MainStage 具備 ReWire 與 MIDI 輸出支援功能，也就是說你用於演出的一切就在你的 Mac 上，讓你完全掌控。每個 Patch 都是功能完備的 mixer 與 router，因此你可以在演出中完全改變風格；這一分鐘你與一個完整的伴奏樂隊一起彈奏重度失真吉他，下一分鐘又可以利用外接 synth 搭配 Ultrabeat 鼓聲模式來創作 bass loop。
